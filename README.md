## Projects

### HTML Advanced

**Directory**: `html_advanced/`  
**Files**: 50+ HTML files, Python scripts, and assets

#### Key Features:
- **Progressive Development**: 40 incremental HTML files (0-39)
- **Semantic HTML5**: Modern markup with proper semantic elements
- **Multimedia Integration**: Video, audio, and iframe implementations
- **Accessibility**: ARIA labels, alt text, and screen reader support
- **SEO Optimization**: Meta tags, structured data, and semantic markup
- **Component Library**: Comprehensive styleguide with reusable components

#### Main Pages:
- **Homepage** (`index.html` â†’ `29-index.html`): Complete website with all sections
- **About** (`about.html`): Company information and team
- **Contact** (`contact.html`): Contact form and information
- **Latest News** (`latest_news.html`): Blog and news articles
- **Article** (`article.html`): Individual article template
- **Styleguide** (`styleguide.html`): Component documentation

#### Styleguide Components:
- Typography (H1-H6, paragraphs, lists)
- Tables with structured data
- Forms and input elements
- Multimedia elements (video, audio)
- Blockquotes and citations
- Interactive elements (details/summary)

### CSS Advanced

**Directory**: `CSS_advanced/`  
**Files**: 32 CSS files, 11 image assets

#### Key Features:
- **Progressive Styling**: 32 incremental CSS files (1-32)
- **Modern CSS**: CSS3 features, custom properties, flexbox, grid
- **Animations & Transitions**: Smooth interactions and visual effects
- **Responsive Design**: Mobile-first approach with flexible layouts
- **Component Styling**: Comprehensive styling for all HTML components
- **Advanced Techniques**: Pseudo-elements, transforms, and animations

#### CSS Progression:
1. **Foundation (1-10)**: Basic styles, typography, custom properties

## Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor or IDE
- Local web server (optional, for testing)

### Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/holbertonschool-web_front_end.git
   cd holbertonschool-web_front_end
   ```

2. **Open in browser**:
   ```bash
   # For HTML Advanced
   open html_advanced/index.html
   
   # For individual files
   open html_advanced/29-index.html
   ```

3. **Local server (recommended)**:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

## Development Process

### HTML Development
1. **Structure First**: Semantic HTML without styling
2. **Content Addition**: Progressive content development
3. **Enhancement**: Adding multimedia and interactive elements
4. **Validation**: W3C HTML validation
5. **Testing**: Cross-browser compatibility

### CSS Development

2. **Layout**: Flexbox and grid implementations
3. **Components**: Individual component styling
4. **Interactions**: Hover states and transitions
